## 使用说明

1. **单条测试**：按序号逐条输入，观察系统响应是否符合预期
2. **链式测试**：Query 1→2→3 模拟售前完整流程；Query 7→8→9→10→11 模拟意图切换+异常恢复流程
3. **情绪测试**：Query 6→7→14→15 覆盖情绪从不满到愤怒的升级路径
4. **兜底测试**：Query 18→19 验证连续模糊意图的兜底机制
