# 如何运行项目

## 快速启动

### 步骤1：双击运行 start.py
在项目根目录中，找到 `start.py` 文件，双击运行即可。

---

### 或者手动启动

#### 第一步：启动后端
1. 打开命令行
2. 进入 `backend` 文件夹
3. 运行：`python -m uvicorn app:app --host 0.0.0.0 --port 8001`

#### 第二步：启动前端（新打开一个命令行窗口）
1. 进入 `frontend` 文件夹
2. 运行：`npm start`

#### 第三步：打开浏览器
访问：http://localhost:3000

---

## 环境变量配置

可通过以下环境变量自定义运行参数：

| 变量名 | 默认值 | 说明 |
|--------|--------|------|
| `SERVER_PORT` | 8001 | 后端服务端口 |
| `FRONTEND_PORT` | 3000 | 前端开发服务器端口 |
| `API_BASE_URL` | http://localhost:8001 | 测试脚本使用的API地址 |
| `KIMI_API_KEY` | - | Kimi API密钥（必填） |
| `KIMI_API_BASE` | https://api.moonshot.cn/v1 | Kimi API地址 |
| `KIMI_MODEL` | moonshot-v1-8k | 使用的模型名称 |

## 注意事项

- 后端默认运行在 **8001端口**，可通过 `SERVER_PORT` 环境变量修改
- 前端开发服务器默认运行在 **3000端口**
- 如果需要停止，在命令行按 `Ctrl+C`
